# Как участвовать

Спасибо, что помогаете выращивать реестр `Oxalis triangularis`.

## Что можно предложить

- новую запись о сорте/форме;
- исправление существующей записи;
- фотографию с ясной лицензией;
- источник или ссылку;
- замечание о дубле;
- улучшение сайта или правил.

## Путь вклада

### Если вы не пользуетесь Git

1. Создайте GitHub Issue или отправьте данные через форму, когда она появится.
2. Заполните поля по возможности: название, описание, источник, фото, автор фото, лицензия.
3. Модератор задаст уточняющие вопросы или подготовит PR.

### Если вы пользуетесь Git

1. Создайте ветку.
2. Измените данные или документацию.
3. Запустите `npm run check`.
4. Откройте Pull Request.

## Требования к данным

- Не утверждайте больше, чем можете подтвердить.
- Для спорных данных используйте статус `uncertain`.
- Для устаревших названий — `deprecated`.
- Для фото обязательно указывайте автора и лицензию.
- Не загружайте медиа прямо в репозиторий без согласования.

## Стиль общения

- Критика направлена на запись, не на человека.
- Уточнение — не нападение.
- Спорная запись лучше временно остаётся `uncertain`, чем публикуется как факт.

## Малые изменения лучше больших

Предпочтительно:

- один PR — одна запись или одна тема;
- небольшие изменения правил;
- ясное описание мотивации.

Так сад легче полоть и труднее случайно повредить.
