Главная Вклад Правила Модерация Roadmap

Модерация

Проверка без лишней тяжести

Сначала автоматика помогает, потом человек решает. Не наоборот.

Автоматическая проверка

  • схема данных;
  • дубли;
  • отсутствие лицензий;
  • большие файлы.

Человеческий review

  • источники;
  • названия;
  • тона текста;
  • спорные случаи.

Эскалация

  • простые вопросы — модератор;
  • спорные данные — куратор;
  • конфликт — два модератора.

Цель

Сделать принятие решений предсказуемым

Чем яснее очередь и статусы, тем меньше напряжение и для участников, и для модераторов.

Хороший ответ

«Спасибо, запись выглядит полезной. Нужен ещё один источник и лицензия для фото — после этого можно принять».
