# Постановка проекта

`Oxalis Triangularis Community` — открытое сообщество и простой публичный реестр форм, сортов, наблюдений и материалов об `Oxalis triangularis`.

Проект нужен, чтобы собрать разрозненные знания коллекционеров, ботаников, селекционеров и любителей в понятную, проверяемую и бережно модерируемую систему.

## Проблема

Информация о растениях часто живёт в личных заметках, чатах, постах, старых каталогах и отдельных фотографиях. Из-за этого появляются:

- дубли названий и спорные идентификации;
- записи без источников и статуса уверенности;
- фотографии без понятной лицензии;
- потеря контекста при обсуждениях;
- высокая нагрузка на людей, которые пытаются поддерживать порядок вручную.

## Решение

Мы строим минимальный, прозрачный и устойчивый контур:

1. статический сайт для чтения и навигации;
2. GitHub как источник истории изменений;
3. схему данных для записей о формах, сортах и наблюдениях;
4. мягкую модерацию через Issue/PR-процесс;
5. помощь агентных инструментов для черновиков, проверок и поиска несоответствий;
6. человеческое решение по публикации и спорным вопросам.

## Границы MVP

В первой версии проект не пытается быть социальной сетью, маркетплейсом или тяжёлой базой данных. MVP должен:

- объяснять назначение сообщества;
- принимать предложения через понятный процесс;
- хранить структуру данных и правила модерации;
- показывать базовый сайт без сложной сборки;
- оставаться понятным новому участнику за несколько минут.

## Критерий успеха

Проект успешен, если участник может предложить полезную запись, модератор может быстро понять её качество, а читатель видит источники, статус уверенности и историю изменений без лишней инфраструктуры.
